# BIP38
This is a fork from [decentropy/bip38](https://github.com/decentropy/bip38) updated to **Python 3.10** and an additional support for other coins.

This implementation supports
* BTC (mainnet / testnet)
* LTC (mainnet / testnet)
* DOGE (mainnet / testnet)
